# tack_copy_mini_appli
# コピペ収集アプリ (Clipboard Collector)

HSP3で作成されていたクリップボード監視・連結アプリを、より堅牢で動作が安定したPython (Tkinter) へ移植したプロジェクトです。

---

## ?? 概要

本アプリケーションは、クリップボード（コピー操作）をリアルタイムに自動監視し、コピーされたテキストを自動的に下部へ、改行を挟みながらどんどん蓄積していく執筆・編集支援ツールです。

HSP3版で課題となっていた「連続コピー時のハングアップ」や「特定データコピー時の強制終了」といったエラーに対し、Python独自の例外ハンドリングを適用することで、いかなる場面でも「絶対に落ちない」高い安定性を実現しました。

---

## ?? 主な改善点と特徴

* **堅牢なエラーハンドリング（例外処理）**
  * テキスト以外のデータ（画像やファイル等）がコピーされた場合や、システムの競合によりクリップボードが一瞬ロックされた場合でも、処理を優しく受け流して監視を継続します。
* **無制限のデータ蓄積**
  * メモリ容量を事前に確保する必要があったHSP3の仕様（`sdim`）から脱却し、Pythonがメモリ割り当てを動的に完全管理。数十万字、数百万字に及ぶ大容量の文章連結でもクラッシュしません。
* **スマート・オートスクロール**
  * 新しい文章が追加されると、自動的に下部のテキストエリアが最下部までスクロールし、執筆中の作業視線を遮りません。
* **低負荷かつ高速なリアルタイム監視**
  * `goto` による無限ループ負荷を排除し、Tkinterが提供する非同期タイマーイベント（`after`）によるミリ秒単位の監視スケジュール制御を行います。

## ?? 動作環境・インストール

本アプリの動作には、お使いの環境にPython 3がインストールされている必要があります。

1. **Python 3 のインストール**
   * [Python公式サイト](https://www.python.org/) から最新版をダウンロードし、インストールしてください。※インストール時に「Add Python to PATH」にチェックを入れてください。
2. **スクリプトの保存**
   * 提供されたPythonコードを `clipboard_app.py` というファイル名で保存します。
3. **起動方法**
   * ターミナル（Mac）またはコマンドプロンプト（Windows）を起動し、保存したフォルダに移動して以下のコマンドを実行します。
     ```bash
     python clipboard_app.py
     ```

---

画面の構成と操作方法
画面は「左側の各種ボタン」と「右側の２つの入力欄（上・下）」で構成されています。

1. 右側の入力欄
上の入力欄（入力欄A）
最新のコピー（複製）された文章がここに表示されます。

下の入力欄（入力欄B）
これまでにコピーされた文章が、２行の改行を挟んでどんどん下に積み重なって表示されます。

2. 左側のボタン機能
copy (手動追加)
自動監視だけでなく、上の入力欄にある文章を手動で下の大きな入力欄に追加したいときに押します。

restart (再起動/初期化)
監視状況を一度きれいに整理し、最新のコピーから改めて拾い直すための準備を行います（今までの文章は消えません）。

chk up / chk down (予備)
今後の機能追加などのために用意されている予備のボタンです。

clear (全消去)
上の入力欄、下の入力欄、そしてこれまでに溜めた文章をすべてきれいに消去し、最初からやり直します。

copy ALL (全体複製)
下の大きな入力欄に溜まったすべての文章を、上の入力欄とクリップボードに同期し直します。これにより、溜まった文章を一発で別の場所に貼り付けられるようになります。
